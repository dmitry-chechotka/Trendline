# Trendline

Trendline ir tīmekļa lietojumprogramma svara izsekošanai, izstrādāta ar HTML, CSS un JavaScript, izmantojot Firebase autentifikācijai un datu glabāšanai.

## Par projektu

Trendline ir paredzēta sportistiem un fiziski aktīviem cilvēkiem, kuri vēlas izsekot savam svaram laika gaitā un saņemt uz datiem balstītus ieskatus un ieteikumus. Tā nav tikai izsekošanas lietotne — tā ir analīzes rīks, kas palīdz lietotājiem saprast savu progresu un sasniegt savus mērķus.

## Funkcijas

- Lietotāja reģistrācija un pieteikšanās caur Firebase Authentication
- Personīgais dashboard ar svara kopsavilkumu un progresa grafiku
- Svara ievades dialogs svara ieloģošanai ar datumu
- Profila lapa personīgo datu rediģēšanai
- Iestatījumu lapa svara mērķu noteikšanai un prognozes apskatei
- Analīze un personalizēti ieteikumi balstoties uz lietotāja datiem
- Aizsargātas lapas — neautentificēts lietotājs tiek novirzīts uz sākumlapu

## Izmantotās tehnoloģijas

- HTML, CSS, JavaScript
- Firebase Authentication
- Firebase Firestore

# EN - Trendline

Trendline is a web-based weight tracking application built with HTML, CSS, and JavaScript, using Firebase for authentication and data storage.

## About

Trendline is designed for athletes and physically active people who want to track their weight over time and receive data-based insights and recommendations. It is not just a tracker — it is an analysis tool that helps users understand their progress and reach their goals.

## Features

- User registration and login via Firebase Authentication
- Personal dashboard with weight summary and progress graph
- Weight entry dialog for logging weight with date
- Profile page for editing personal data
- Settings page for setting weight goals and viewing prognosis
- Analysis and personalized insights based on user data
- Protected pages — unauthenticated users are redirected to the homepage

## Technologies Used

- HTML, CSS, JavaScript
- Firebase Authentication
- Firebase Firestore
